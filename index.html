How to get full data for all 750 securities
  1. In YCharts, open a Comp Table
  2. Import the full ticker list from this universe (or paste your target list)
  3. Add columns: P/E Ratio, Price to Book Value, Free Cash Flow Yield, Debt to Equity, Revenue (QoQ YoY Growth), Net Income (QoQ YoY Growth), PEG Ratio
  4. Enable AutoUpdate, then export as CSV
  5. Upload the CSV using the button in the top-right header
  6. The dashboard will automatically detect and map all columns — all 750 scores will update instantly
